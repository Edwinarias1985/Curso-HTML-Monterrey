Formato y estructura de HTML

HTML es el esqueleto de todas las páginas WEB es decir que, aquí se puede definir el formato, perimite incluir imágenes, formularios, botones e incluso enlaces. Nos permite usar todas las herramientas para hacer uso de nuestra imaginación.

Lo que se hace en HTML es lo que nuestros navegadores lo interpretan y nos dejan ver como finalmente queda

Competencias que se adquiere en este módulo

  • Reglas generales de HTML.
  • Conocimientos sobre elementos de HTML como encabezados, listas, párrafos, formularios.
  • Estructuramiento básico de páginas WEB.
