Tema 12: Enlaces Internos

Enlaces a otras páginas del mismo sitio

Este tipo de enlaces se usa todo el timepo para navegar en el sitio WEB, en la barra de navegación. En el atributo href se debe colocar ya no la URL sino la ruta al archivo al que queramos llamar, navegar.

Enlaces a otras partes de la misma página

Estos enlaces se utilizan para facilitar la navegación al usuario, al presionarlo lo lleva directamente al inicio o final de una sección y no utilizar el desplazamiento del Mouse. Para esto en el atributo href se le debe poner # y en el id pues ya creado anteriormente en alguna etiqueta, se le llama con el nombre que se le haya puesto, y así poder utilizarlo con el # en el atribito href, recordar que en el atributo global id se puede utilizar en cualquier elemento HTML.

Otros elementos

Con este módulo se llega al final de HTML y así, se conocieron todos los elementos necesarios para esturcturar una página WEB. Para esta sección se conocerán otros elementos de HTML que podrán ser mu útiles como lo son: iframes, tablas y spans.

Iframes

Los iframes son visualizados como un marco que puede contener otra página WEB, tanto interna como externa en nuestro sitio. Como ejemplo es videos de Youtube, para poder implementar estos elementos se utiliza la etiqueta iframe y para utilizarlo se necesita definir el atributo src indicando al navegador el URL de la página que debe mostrar dentro del marco. Tablas

Se definen con la etiqueta table donde las tablas nos permiten organizar contenido en columnas y filas. Para esto se utiliza las etiquetas:

  • tr: para definir filas
  • th: para definir encabezados de tablas
  • td: para definir celdas normales
