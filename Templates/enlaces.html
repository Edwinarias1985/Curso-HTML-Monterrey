Tema 11: enlaces

Son llamados como enlaces o hipervinculos, éstos son muy comunes en todas las páginas WEB donde nos permiten navegar tanto navegadores externos, como en archivos de nuestro PC. Para esto se debe utilizar la etiqueta (a). No necesariamente sirve para texto de una página sino, tambien para videos, imágenes y URL's. También se puede utilizar en cualquier elemento HTML.
