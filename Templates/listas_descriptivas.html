Listas descriptivas en HTML (HyperText Markup Language)

Es este tipo de lista es muy común con las listas Ordenadas y Desordenadas solo que aquí las etiquetas son diferentes, se tiene la etiqueta padre dl, las etiquetas hijas son dt la cual es para definir términos, y la dd es para agregar descripción de ellos. Se puede ver como una aplicación común de estos elementos en crear un glosario en una página WEB.

Ejemplo

Café
Bebida caliente oscura
Leche
Bebida fría de color balnco
