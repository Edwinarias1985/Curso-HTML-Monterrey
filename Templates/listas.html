Listas en HTML (Hypertex markup language)

Las listas se pueden utilizar para enumerar elementos o en listarlos con viñetas. Hay tes tipos de listas qu son ordenadas y desordenadas, y cada una cuenta con su propia estrutura que se verá a continuación.

Listas ordenadas

Como su nombre lo indica, muestra sus elementos en orden siguiendo la semántica con su etiqueta ol y por cada elemento se añade la etiqueta li. El atributo TYPE especifíca el tipo de numeracón que le queremos dar a la lista ordenada

Ejemplo:

  1. Primer elemento
  2. segundo elemento
  3. Tercer elemento
  4. Cuarto elemento
    1. Otro primer elemento
    2. Otro segundo elemento

Se puede notar al utilizar dos listar una dentro de otra, se cambió el atributo TYPE para que nos muestre lista con incisos alfabéticos. ya que este atrubito por defecto en numérico

Listas desordenadas.

Los elementos de esta lista no se ORDENAN, su etiqueta es ul y al igual que las listas ordenadas, cada elemento debe estar dentro de la etiqueta li. Este tipo de de lista no tiene atributo type, para modificar su estilo podemos utilizar el atributo style pero, es mejor utilizar un archivo CSS aparte para mantener el orden de los archivos.

Ejemplo:

  • Café
  • Leche
  • Pan
  • Huevos
  • Verdura
    • Lechuga
    • Tomate
    • Cebolla

Como se puede ver en el ejemplo, no se ve que haya numeración en las lista y también se utilizó una lista dentro de otra.
