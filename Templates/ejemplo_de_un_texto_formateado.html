Programación WEB

Es una disciplina que se ha popularizado en los últimos años. El incremnto del Internet ha permitido la creación de millónes de páginas WEB

Principales lenguajes

Estos lenguajes cumplen una función específica (HTML, CSS y JavaScript)

  • HTML

    • Este lenguaje le da estructura a las páginas.

  • CSS

    • Este lenguaje es para decorar páginas.

  • JavaScript

    • Este lenguaje es para manipular elementos de las páginas.
